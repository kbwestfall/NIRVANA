====================================================================================
NIRVANA:  **N**onaxisymmetric **I**rregular **R**otational **V**elocity **ANA**lysis
====================================================================================

NIRVANA forward models disk galaxy kinematic data.  It also accounts for beam
smearing and velocity dispersion in the resolved spectra. 

Installation
============

- Fork and clone the repository

- From the top-level directory:

..code-block::

    pip install -e ".[dev]"
